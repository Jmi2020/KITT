# KITTY Knowledge Base

This directory contains structured knowledge that KITTY can read and autonomously update.

## Directory Structure

### `materials/`
Specifications and properties for 3D printing materials (filaments, resins, powders).

**Format**: Markdown with YAML frontmatter
**Naming**: `{material-slug}.md` (e.g., `pla.md`, `petg-recycled.md`)
**Frontmatter fields**:
- `cost_per_kg`: Number (USD/kg)
- `density`: Number (g/cm³)
- `print_temp`: Number or range (°C)
- `bed_temp`: Number or range (°C)
- `sustainability_score`: Number (1-10, higher = more sustainable)
- `suppliers`: Array of supplier names

### `techniques/`
Troubleshooting guides and best practices for 3D printing.

**Format**: Markdown
**Naming**: `{technique-slug}.md` (e.g., `first-layer-adhesion.md`, `support-optimization.md`)
**Sections**: Problem, Causes, Solutions, References

### `equipment/`
Documentation for printers, devices, and maintenance procedures.

**Structure**:
- `printers/{printer-id}.md` - Individual printer specs
- `printers/capabilities.md` - Fleet capabilities summary
- `maintenance/calibration-schedule.md` - Maintenance procedures

### `research/`
Autonomous research articles generated by KITTY.

**Naming convention**: `YYYY-Www-topic-slug.md` (e.g., `2025-W19-sustainable-filament-suppliers.md`)
**Frontmatter fields**:
- `generated_date`: ISO 8601 timestamp
- `goal_id`: UUID of associated Goal
- `project_id`: UUID of associated Project
- `cost_usd`: Research API cost
- `sources`: Array of URLs

## Usage

KITTY's autonomous system accesses this knowledge base via:
- `KnowledgeDatabase` (read operations): query, search, retrieve
- `KnowledgeUpdater` (write operations): create, update, git commit

All updates are automatically committed to version control with message prefix `KB: autonomous update`.

## Maintenance

- Manual additions are welcome (follow format conventions)
- Do not delete autonomously-generated research articles without archiving
- Update frontmatter when material costs or specs change
